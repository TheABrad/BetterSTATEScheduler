About BetterSTATEScheduler

This site was created to be a better version of the PSU Schedule of Courses website. I was inspired to do this by the designs of Hipmunk and BetterMetroNorth

Usage

Select a course subject by typing it into the search bar, pick which semester you are looking for, and click the submit button. Currently this search only works for the University Park campus.

GitHub

The source for BetterSTATEScheduler can be found on GitHub. Feel free to send a pull request or submit an issue.
